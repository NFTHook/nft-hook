
# NFT-HOOK

## Project Overview

NFT-HOOK is the backend service for NFTHook.

### Prerequisites

*   Go (version 1.20 or higher)
*   Linux environment

### Build

CGO_ENABLED=0 GOOS=linux GOARCH=amd64 go build -o nfthook-serv main.go
